# Codeword Puzzle Game

A fun React TypeScript word puzzle game where players decode words by figuring out letters that have been shifted by one position in the alphabet. **Now available as a native Android app!**

## Game Rules

- Each round presents an encoded word where every letter has been shifted forward by 1 position in the alphabet
- For example: A becomes B, B becomes C, Z becomes A
- Players must figure out the original word and enter it in the input fields
- Words are between 3-6 letters long from a curated catalog
- Click "Check Answer" to see if you guessed correctly!

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn
- **For Android builds**: Android Studio and Android SDK

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd codeword-puzzle
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Building for Android 📱

### Quick Build (Automated)

Use the provided build script for a one-command build:

```bash
chmod +x build-android.sh
./build-android.sh
```

This script will:
- Install dependencies
- Build the React app
- Set up Capacitor
- Generate the Android project
- Build the APK

### Manual Build Process

1. **Build the web app:**
```bash
npm run build
```

2. **Sync with Capacitor:**
```bash
npm run build:mobile
```

3. **Open in Android Studio:**
```bash
npx cap open android
```

4. **Build APK in Android Studio:**
   - Click "Build" → "Build Bundle(s) / APK(s)" → "Build APK(s)"
   - Or use Gradle: `cd android && ./gradlew assembleDebug`

### Available Mobile Scripts

- `npm run build:mobile` - Builds web app and syncs with Capacitor
- `npm run android:dev` - Runs app on connected Android device/emulator
- `npm run android:build` - Builds production APK

### APK Location

After successful build, find your APK at:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

### Installing on Device

```bash
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

## How It Works

1. **Word Selection**: A random word is selected from a catalog of 3-6 letter words
2. **Encoding**: Each letter is shifted forward by 1 position (Caesar cipher with shift +1)
3. **Display**: The encoded word is shown to the player
4. **Input**: Player enters their guess letter by letter with smart navigation
5. **Evaluation**: The game checks if the guess matches the original word
6. **Feedback**: Win/lose message is displayed with haptic feedback and option to play again

## Troubleshooting

### Common Build Issues

**Capacitor not found:**
```bash
npm install @capacitor/cli @capacitor/core @capacitor/android
```

**Android SDK not found:**
- Install Android Studio
- Set ANDROID_HOME environment variable
- Add Android SDK tools to PATH

**Gradle build fails:**
- Open project in Android Studio
- Let it sync and download dependencies
- Build from IDE or use `./gradlew assembleDebug`

**APK not installing:**
- Enable "Unknown sources" in Android settings
- Use `adb install -r` to replace existing app
